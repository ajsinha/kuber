Managing Users

Users are managed through the users.json file. To add, modify, or remove users:

  1. Edit the users.json file in the application's resources directory
  2. Add or modify user entries with userId, password, fullName, and roles
  3. Click the "Reload Users" button or restart the application

Example user entry:

{
  "userId": "newuser",
  "password": "password123",
  "fullName": "New User",
  "roles": ["USER"]
}
