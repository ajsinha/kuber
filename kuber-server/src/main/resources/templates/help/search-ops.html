Search Operations

KEYS vs KSEARCH

FeatureKEYSKSEARCH
PatternGlob (user:*)Regex (user:\d+)
ReturnsKey names onlyKeys + values + metadata
Use caseSimple patternsComplex matching

KSEARCH Examples

KSEARCH user:\d+           # user:1, user:2, etc
KSEARCH order:2024-.*      # Orders from 2024
KSEARCH session:[a-f0-9]+  # Hex session IDs
