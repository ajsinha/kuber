Redis Protocol

Version 1.7.8: Kuber supports both standard Redis commands and Kuber-specific extensions. Connect via Redis protocol on port 6380 (default) and authenticate with AUTH kub_your_api_key.

Complete Command Reference

The table below lists all supported commands. Commands marked as Kuber Extension are unique to Kuber and not available in standard Redis.

Command Syntax Description Type
Authentication
AUTH AUTH api_key Authenticate with API key. Required before any other command. Redis Compatible
PING PING [message] Test connection. Returns PONG or echoes the message. Redis Compatible
String Operations
GET GET key Get the value of a key. Returns nil if key doesn't exist. Redis Compatible
SET SET key value [EX seconds] [PX ms] [NX|XX] Set key to value. Options: EX/PX for TTL, NX only if not exists, XX only if exists. Redis Compatible
SETEX SETEX key seconds value Set key with expiration time in seconds. Redis Compatible
SETNX SETNX key value Set key only if it doesn't exist. Returns 1 if set, 0 if already exists. Redis Compatible
GETSET GETSET key value Set key and return old value atomically. Redis Compatible
APPEND APPEND key value Append value to existing key. Creates key if doesn't exist. Redis Compatible
STRLEN STRLEN key Get the length of the value stored at key. Redis Compatible
INCR INCR key Increment integer value by 1. Creates key with 0 if doesn't exist. Redis Compatible
INCRBY INCRBY key increment Increment integer value by specified amount. Redis Compatible
DECR DECR key Decrement integer value by 1. Redis Compatible
DECRBY DECRBY key decrement Decrement integer value by specified amount. Redis Compatible
Key Operations
DEL DEL key [key ...] Delete one or more keys. Returns number of keys deleted. Redis Compatible
EXISTS EXISTS key [key ...] Check if key(s) exist. Returns count of existing keys. Redis Compatible
KEYS KEYS pattern Find all keys matching glob pattern (e.g., user:*). Redis Compatible
SCAN SCAN cursor [MATCH pattern] [COUNT count] Incrementally iterate over keys. Cursor-based for large datasets. Redis Compatible
TYPE TYPE key Get the type of value stored at key (string, hash, list, etc.). Redis Compatible
RENAME RENAME key newkey Rename a key. Overwrites newkey if it exists. Redis Compatible
RENAMENX RENAMENX key newkey Rename key only if newkey doesn't exist. Redis Compatible
DBSIZE DBSIZE Return the number of keys in the current region. Redis Compatible
FLUSHDB FLUSHDB Delete all keys in the current region. Redis Compatible
KSEARCH KSEARCH pattern [LIMIT count] Search keys using regex pattern, returns keys with values. Kuber Extension
TTL Operations
TTL TTL key Get remaining time to live in seconds. -1 if no expiry, -2 if key doesn't exist. Redis Compatible
PTTL PTTL key Get remaining time to live in milliseconds. Redis Compatible
EXPIRE EXPIRE key seconds Set expiration time in seconds from now. Redis Compatible
PEXPIRE PEXPIRE key milliseconds Set expiration time in milliseconds from now. Redis Compatible
EXPIREAT EXPIREAT key timestamp Set expiration as Unix timestamp (seconds). Redis Compatible
PERSIST PERSIST key Remove expiration from key, making it persist indefinitely. Redis Compatible
Batch Operations
MGET MGET key [key ...] Get values of multiple keys in a single call. Redis Compatible
MSET MSET key value [key value ...] Set multiple key-value pairs atomically. Redis Compatible
MSETNX MSETNX key value [key value ...] Set multiple keys only if none of them exist. Redis Compatible
Hash Operations
HGET HGET key field Get the value of a hash field. Redis Compatible
HSET HSET key field value [field value ...] Set one or more hash fields. Redis Compatible
HSETNX HSETNX key field value Set hash field only if it doesn't exist. Redis Compatible
HMGET HMGET key field [field ...] Get values of multiple hash fields. Redis Compatible
HMSET HMSET key field value [field value ...] Set multiple hash fields (deprecated, use HSET). Redis Compatible
HGETALL HGETALL key Get all fields and values in a hash. Redis Compatible
HDEL HDEL key field [field ...] Delete one or more hash fields. Redis Compatible
HEXISTS HEXISTS key field Check if hash field exists. Redis Compatible
HKEYS HKEYS key Get all field names in a hash. Redis Compatible
HVALS HVALS key Get all values in a hash. Redis Compatible
HLEN HLEN key Get the number of fields in a hash. Redis Compatible
HINCRBY HINCRBY key field increment Increment hash field integer value. Redis Compatible
List Operations
LPUSH LPUSH key value [value ...] Push values to the head (left) of a list. Redis Compatible
RPUSH RPUSH key value [value ...] Push values to the tail (right) of a list. Redis Compatible
LPOP LPOP key [count] Remove and return element(s) from the head of a list. Redis Compatible
RPOP RPOP key [count] Remove and return element(s) from the tail of a list. Redis Compatible
LRANGE LRANGE key start stop Get a range of elements from a list (0-indexed, -1 for last). Redis Compatible
LLEN LLEN key Get the length of a list. Redis Compatible
LINDEX LINDEX key index Get element at index in a list. Redis Compatible
LSET LSET key index value Set the value at index in a list. Redis Compatible
Set Operations
SADD SADD key member [member ...] Add one or more members to a set. Redis Compatible
SREM SREM key member [member ...] Remove one or more members from a set. Redis Compatible
SMEMBERS SMEMBERS key Get all members of a set. Redis Compatible
SISMEMBER SISMEMBER key member Check if member exists in set. Returns 1 or 0. Redis Compatible
SCARD SCARD key Get the number of members in a set. Redis Compatible
JSON Operations (Kuber Extension)
JSET JSET key json [path] [ttl] Store a JSON document. Path defaults to $ (root). Alias: JSON.SET Kuber Extension
JGET JGET key [path] Get JSON document or value at path. Alias: JSON.GET Kuber Extension
JUPDATE JUPDATE key json [ttl] Merge JSON with existing value (upsert). Deep merges nested objects. New values override existing. Alias: JSON.UPDATE Kuber Extension
JREMOVE JREMOVE key ["attr1","attr2",...] Remove specified attributes from JSON document. Does nothing if key missing or not JSON object. Alias: JSON.REMOVE Kuber Extension
JDEL JDEL key [path] Delete JSON document or value at path. Alias: JSON.DEL Kuber Extension
JSEARCH JSEARCH query [LIMIT count] Search JSON documents by field values. Query format: $.field=value, $.field>value, etc. Alias: JSON.SEARCH Kuber Extension
Region Operations (Kuber Extension)
RSELECT RSELECT region Select a region for subsequent operations. Like Redis SELECT for databases. Kuber Extension
RCREATE RCREATE name [description] Create a new region with optional description. Kuber Extension
RDROP RDROP region Delete a region and all its data. Kuber Extension
RPURGE RPURGE region Delete all data in a region but keep the region. Kuber Extension
REGIONS REGIONS List all available regions with metadata. Kuber Extension
RSET RSET region key value [ttl] Set a value in a specific region without RSELECT. Kuber Extension
RGET RGET region key Get a value from a specific region without RSELECT. Kuber Extension
RSETMAP RSETMAP region key field value [field value ...] Set hash fields in a specific region (maps). Kuber Extension
RGETMAP RGETMAP region key [field ...] Get hash fields from a specific region. Kuber Extension
Server Operations
INFO INFO [section] Get server information and statistics. Redis Compatible
STATUS STATUS Get Kuber server status including regions, memory, and connections. Kuber Extension
REPLINFO REPLINFO Get replication status and cluster information. Kuber Extension
QUIT QUIT Close the connection. Redis Compatible
SELECT SELECT db Select database (Kuber maps to region index). Redis Compatible
ECHO ECHO message Echo the given message. Redis Compatible
Connection Management v1.7.8
CLIENT LIST CLIENT LIST List all connected clients with their details. Redis Compatible
CLIENT ID CLIENT ID Get the current client's session ID. Redis Compatible
CLIENT INFO CLIENT INFO Get detailed information about the current client connection. Redis Compatible
CLIENT GETNAME CLIENT GETNAME Get the client name set by CLIENT SETNAME. Redis Compatible
CLIENT SETNAME CLIENT SETNAME name Set a name for the current connection (useful for debugging). Redis Compatible
CLIENT GETTIMEOUT CLIENT GETTIMEOUT Get the session idle timeout in seconds. Kuber Extension
CLIENT SETTIMEOUT CLIENT SETTIMEOUT seconds Set session timeout (0=no timeout, 60-86400 seconds). Prevents connection from timing out. Kuber Extension
CLIENT KEEPALIVE CLIENT KEEPALIVE Reset idle timer, returns PONG. Use to keep long-running connections alive. Kuber Extension
CLIENT HELP CLIENT HELP Show available CLIENT subcommands. Redis Compatible
Note: Some Redis commands may have slight behavioral differences in Kuber. Kuber Extension commands are unique to Kuber and provide additional functionality beyond standard Redis.
