Autoload - Bulk Import

Overview

Import data from CSV and JSON files by placing them in the autoload directory.

Directory Structure

data/autoload/
├── inbox/          # Place files here
│   ├── data.csv
│   └── data.csv.metadata
└── outbox/         # Processed files moved here

Metadata File Format

region=customers
key_field=customer_id
ttl=3600

Attribute Mapping

Create data.csv.metadata.attributemapping.json:

{
    "firstName": "first_name",
    "lastName": "last_name"
}
