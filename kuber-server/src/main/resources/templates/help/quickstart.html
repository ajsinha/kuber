Quick Start

1. Start the Server

java -jar kuber-server-1.1.3.jar

2. Connect with Redis CLI

redis-cli -p 6379 -a admin123
127.0.0.1:6379> PING
PONG
127.0.0.1:6379> SET user:1001 "John Doe"
OK
127.0.0.1:6379> GET user:1001
"John Doe"

3. Use REST API

# Store a value
curl -X PUT -u admin:admin123 \
  -H "Content-Type: application/json" \
  -d '{"name":"John","status":"active"}' \
  http://localhost:8080/api/cache/default/user:1001

# Get a value
curl -u admin:admin123 http://localhost:8080/api/cache/default/user:1001

4. Access Web UI

Open http://localhost:8080 and login with admin / admin123

Default Ports: REST API on 8080, Redis protocol on 6379
