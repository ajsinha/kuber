Generic Search API

Overview

Unified search API supporting key patterns, JSON queries, and field projection.

Request Body

{
    "region": "users",           // Required
    "key": "user:1001",          // Mode 1: Exact key
    "keypattern": "user:\\d+",  // Mode 2: Regex pattern
    "type": "json",              // Mode 3: JSON search
    "values": [{"status": "active"}],  // JSON conditions
    "fields": ["name", "email"], // Field projection
    "limit": 100                 // Max results
}

Search Modes

  • Key lookup: Single key retrieval
  • Key pattern: Regex-based key search
  • JSON search: Query by attribute values
  • Field projection: Return only specified fields
