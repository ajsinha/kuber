REST API Reference

Authentication

Use HTTP Basic Auth: Authorization: Basic base64(username:password)

Or use API Key: X-API-Key: kub_your_api_key

v2.1.0: Role-Based Access Control
All endpoints require appropriate permissions. Operations return HTTP 403 if permission is denied. See Security & RBAC for details.
