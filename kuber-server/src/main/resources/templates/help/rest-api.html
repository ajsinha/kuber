REST API Reference

Authentication

Use HTTP Basic Auth: Authorization: Basic base64(username:password)

Endpoints

MethodEndpointDescription
GET/api/infoServer information
GET/api/regionsList all regions
POST/api/regionsCreate region
GET/api/cache/{region}/{key}Get value
PUT/api/cache/{region}/{key}Set value
DELETE/api/cache/{region}/{key}Delete key
POST/api/genericsearchGeneric search
POST/api/cache/{region}/jsearchJSON search

See Python Client or Java Client for complete examples.
