String Operations

String operations are the foundation of key-value caching. These commands allow you to store, retrieve, and manipulate string values.

Command Reference
CommandSyntaxDescription
SETSET key value [EX seconds] [PX ms] [NX|XX]Set string value with optional TTL and conditions
GETGET keyGet the value of a key
SETEXSETEX key seconds valueSet value with expiration in seconds
SETNXSETNX key valueSet only if key does not exist
GETSETGETSET key valueSet new value and return old value
APPENDAPPEND key valueAppend value to existing string
STRLENSTRLEN keyGet length of string value
INCRINCR keyIncrement integer value by 1
INCRBYINCRBY key incrementIncrement integer value by amount
DECRDECR keyDecrement integer value by 1
DECRBYDECRBY key decrementDecrement integer value by amount
SET - Set String Value

Set the string value of a key. If the key already exists, it is overwritten.

  • Redis CLI
  • Python
  • Java
  • C#
# Basic SET
SET greeting "Hello, World!"

# SET with expiration (60 seconds)
SET session:token "abc123" EX 60

# SET only if not exists (NX)
SET lock:resource "locked" NX

# SET only if exists (XX)
SET counter "100" XX
with KuberRedisClient('localhost', 6380, api_key='kub_xxx') as client:
    # Basic SET
    client.set('greeting', 'Hello, World!')
    
    # SET with TTL (60 seconds)
    client.set('session:token', 'abc123', ttl=60)
    
    # SET only if not exists
    result = client.setnx('lock:resource', 'locked')
    print(f"Lock acquired: {result}")  # True if set
try (KuberClient client = new KuberClient("localhost", 6380, "kub_xxx")) {
    // Basic SET
    client.set("greeting", "Hello, World!");
    
    // SET with TTL (60 seconds)
    client.setex("session:token", 60, "abc123");
    
    // SET only if not exists
    boolean acquired = client.setnx("lock:resource", "locked");
}
using var client = new KuberClient("localhost", 6380, "kub_xxx");

// Basic SET
await client.SetAsync("greeting", "Hello, World!");

// SET with TTL (60 seconds)
await client.SetAsync("session:token", "abc123", TimeSpan.FromSeconds(60));

// SET only if not exists
bool acquired = await client.SetNxAsync("lock:resource", "locked");
GET - Get String Value

Get the value of a key. Returns nil if the key does not exist.

  • Redis CLI
  • Python
  • Java
  • C#
GET greeting
# Returns: "Hello, World!"

GET nonexistent
# Returns: (nil)
value = client.get('greeting')
print(f"Value: {value}")  # "Hello, World!"

value = client.get('nonexistent')
print(f"Value: {value}")  # None
String value = client.get("greeting");
System.out.println("Value: " + value);  // "Hello, World!"

String missing = client.get("nonexistent");
System.out.println("Value: " + missing);  // null
string? value = await client.GetAsync("greeting");
Console.WriteLine($"Value: {value}");  // "Hello, World!"

string? missing = await client.GetAsync("nonexistent");
Console.WriteLine($"Value: {missing}");  // null
INCR / INCRBY / DECR / DECRBY - Counter Operations

Atomically increment or decrement integer values. Creates key with 0 if doesn't exist.

  • Redis CLI
  • Python
  • Java
  • C#
SET pageviews 100

INCR pageviews        # Returns: 101
INCRBY pageviews 10   # Returns: 111
DECR pageviews        # Returns: 110
DECRBY pageviews 5    # Returns: 105
client.set('pageviews', '100')

client.incr('pageviews')         # 101
client.incrby('pageviews', 10)   # 111
client.decr('pageviews')         # 110
client.decrby('pageviews', 5)    # 105
client.set("pageviews", "100");

client.incr("pageviews");        // 101
client.incrBy("pageviews", 10);  // 111
client.decr("pageviews");        // 110
client.decrBy("pageviews", 5);   // 105
await client.SetAsync("pageviews", "100");

await client.IncrAsync("pageviews");        // 101
await client.IncrByAsync("pageviews", 10);  // 111
await client.DecrAsync("pageviews");        // 110
await client.DecrByAsync("pageviews", 5);   // 105
APPEND / STRLEN - String Manipulation

Append to strings and get string length.

  • Redis CLI
  • Python
  • Java
  • C#
SET message "Hello"
APPEND message ", World!"   # Returns: 13 (new length)
GET message                 # Returns: "Hello, World!"
STRLEN message              # Returns: 13
client.set('message', 'Hello')
client.append('message', ', World!')  # Returns: 13
client.get('message')                 # "Hello, World!"
client.strlen('message')              # 13
client.set("message", "Hello");
client.append("message", ", World!");  // Returns: 13
client.get("message");                 // "Hello, World!"
client.strlen("message");              // 13
await client.SetAsync("message", "Hello");
await client.AppendAsync("message", ", World!");  // Returns: 13
await client.GetAsync("message");                 // "Hello, World!"
await client.StrLenAsync("message");              // 13
GETSET / SETNX / SETEX - Atomic Operations

Atomic operations for concurrency control and TTL management.

  • Redis CLI
  • Python
  • Java
  • C#
# GETSET - atomically set new, return old
SET counter "100"
GETSET counter "200"    # Returns: "100"
GET counter             # Returns: "200"

# SETNX - set only if not exists (for locks)
SETNX lock:job "process-1"   # Returns: 1 (success)
SETNX lock:job "process-2"   # Returns: 0 (already exists)

# SETEX - set with TTL
SETEX session:token 3600 "abc123"
# GETSET
client.set('counter', '100')
old = client.getset('counter', '200')  # Returns: "100"

# SETNX - acquire lock
acquired = client.setnx('lock:job', 'process-1')  # True

# SETEX - set with TTL
client.setex('session:token', 3600, 'abc123')
// GETSET
client.set("counter", "100");
String old = client.getSet("counter", "200");  // "100"

// SETNX - acquire lock
boolean acquired = client.setnx("lock:job", "process-1");

// SETEX - set with TTL
client.setex("session:token", 3600, "abc123");
// GETSET
await client.SetAsync("counter", "100");
string? old = await client.GetSetAsync("counter", "200");

// SETNX - acquire lock
bool acquired = await client.SetNxAsync("lock:job", "process-1");

// SETEX - set with TTL
await client.SetExAsync("session:token", 3600, "abc123");
