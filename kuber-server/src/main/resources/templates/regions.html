Glossary
Region
A logical namespace for grouping related cache entries. Provides isolation between different data sets.
Captive
A system-protected region that cannot be deleted (e.g., "default" region).
Total Entries
The total number of key-value pairs stored in this region.
Hit Ratio
Percentage of cache lookups that found data (hits) vs total lookups. Higher is better.
Memory
Entries currently held in fast in-memory cache for quick access.
Persistence
Entries stored in the backing persistence store (RocksDB, MongoDB, etc.) for durability.
