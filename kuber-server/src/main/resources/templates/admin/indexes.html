Performance Guide
HASH Index
  • O(1) lookup - Instant equality queries
  • Best for: status = "active"
  • Low-to-medium cardinality fields
BTREE Index
  • O(log n) lookup - Fast range queries
  • Best for: age > 30, date BETWEEN
  • Numeric and date fields
TRIGRAM Index
  • O(trigrams) - Pattern matching
  • Best for: regex and contains searches
  • Email, name pattern searches
PREFIX Index
  • O(prefix len) - Ultra fast
  • Best for: starts-with queries
  • SKUs, phone numbers, zip codes
