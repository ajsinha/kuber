Glossary
PID
Process ID - unique identifier for the running server process.
Heap Memory
JVM memory used for dynamic object allocation. Critical for performance monitoring.
High Watermark
Memory threshold that triggers automatic eviction to prevent OOM.
Low Watermark
Target memory level after eviction completes.
Eviction
Moving entries from memory to persistence to free heap space.
Autoload
Automatic file-based data loading from inbox directory.
Compaction
RocksDB process to merge SST files and reclaim disk space from deleted entries.
Operations/min
Cache activity rate: gets + sets + deletes per minute.
TTL Expiration
Automatic cleanup of expired entries from both memory and persistence.
