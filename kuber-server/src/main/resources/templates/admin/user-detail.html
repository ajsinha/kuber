Role Permissions
ADMIN
Full system access including user management, configuration, and all cache operations.
OPERATOR
Manage regions, insert/delete entries, view statistics. Cannot manage users.
USER
Browse cache, insert and delete entries. Cannot create or delete regions.
READONLY
View-only access. Can browse and query but cannot modify anything.
